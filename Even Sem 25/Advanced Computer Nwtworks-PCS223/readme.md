# PCS223 — Advanced Computer Networks Lab
**Thapar Institute of Engineering and Technology | MTech CSE**

This repository contains all lab assignments for the Advanced Computer Networks course (PCS223). The labs cover practical network analysis using Wireshark and network configuration/routing using Cisco Packet Tracer.

---

## Tools Used
- **Wireshark** — packet capture and protocol analysis
- **Cisco Packet Tracer** — network simulation and router/switch configuration

---

## Assignments

| # | Title | Tool | Key Concept |
|---|-------|------|-------------|
| 1 | Wireshark Packet Capture | Wireshark | Capture live traffic, apply filters (tcp/udp/dns/icmp/ip) |
| 2 | Protocol Analysis | Wireshark | Analyze ICMP, TCP 3-way handshake, DNS, HTTP, ARP |
| 3 | Network Topologies | Packet Tracer | Bus, Ring, Tree topologies using hubs and switches |
| 4 | DHCP Configuration | Packet Tracer | Configure router as DHCP server, auto IP assignment |
| 5 | Static Routing | Packet Tracer | Manually configure routes between two LANs via serial link |
| 6 | RIP Version 2 | Packet Tracer | Dynamic routing with distance-vector protocol |
| 7 | OSPF Routing | Packet Tracer | Link-state routing, SPF algorithm, OSPF cost analysis |

---

## Concepts Covered
Packet capture and analysis · TCP/IP protocol stack · Network topologies · DHCP · Subnetting · Static routing · Dynamic routing (RIP v2, OSPF) · Link State Database · Dijkstra's SPF algorithm · Wildcard masks · OSPF neighbor adjacency
